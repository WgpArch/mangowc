# 🏠 Welcome to my MangoWC Config

A minimalist, scrollable-tiling Wayland compositor setup on Arch Linux. This configuration features a custom **Dual Waybar** setup, utilizing two different weather modules, extensive animation tweaks, and a unique blend of tiling and scroller layouts.

![MangoWC Showcase](screenshots/Mango_Screenshot_06-03-2026_20-17-03.png)

## ✨ Highlights
- **Dual Waybar:** A primary bar using `dwl/tags` and a Ruby weather script, plus an alternative bar using `ext/workspaces` and a feature-rich weather widget with icons.
- **Animations:** Highly customized slide and zoom animations with specific curves for opening, closing, and moving windows.
- **Blur & Shadows:** Configurable blur parameters and drop shadows for floating windows.
- **Layout Flexibility:** Supports tile, scroller, grid, deck, monocle, and more, with per-workspace layout rules.
